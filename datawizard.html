RETIGO Data Wizard

You can use this tool to help get your data into a format that RETIGO can read. It assumes that you are starting with a plain text (ASCII) comma-separated-value (CSV) file with a header line that describes each column. After you load your file, the data wizard will analyze the file, report any problems that it finds, and ask you for the information it needs to try to convert your data into a valid RETIGO file. Once complete, you can save the coverted file for future use with RETIGO. If you have any questions, please contact the RETIGO development team.
